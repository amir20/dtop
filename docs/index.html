Get Started

Choose your preferred installation method

Homebrew
$ brew install dtop
Cargo
$ cargo install dtop
Shell Script
$ curl --proto '=https' --tlsv1.2 -LsSf https://github.com/amir20/dtop/releases/latest/download/dtop-installer.sh | sh
Docker
$ docker run -v /var/run/docker.sock:/var/run/docker.sock -it ghcr.io/amir20/dtop:latest

Then run:

$ dtop_

Keyboard Shortcuts

  • -> Press ? for help
  • -> Use u n c m to sort by uptime, name, CPU, or memory
  • -> Press Enter to view container logs
  • -> Connect to remote hosts with dtop --host ssh://user@host
